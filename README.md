# Laboration 1.2 - Bildredigering

Detta är en minimal startpunkt för labben. Projektet innehåller:

- `index.html` – enkel sida med en <canvas> för att visa en bild
- `styles.css` – grundläggande styling
- `main.js` – minimal kod för att ladda en bild i canvas
- `package.json` – innehåller script för att köra en dev-server med `lite-server`

## Kör lokalt (Windows PowerShell)

1. Installera beroenden:

```
npm install
```

2. Starta dev-servern:

```
npm start
```

3. Öppna http://localhost:3000 i din webbläsare (ska öppnas automatiskt av lite-server).

Byt ut placeholder-filer och bygg vidare med bildbehandlingsfunktioner enligt labbinstruktionerna.
